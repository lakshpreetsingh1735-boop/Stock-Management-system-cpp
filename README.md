#  Stock Management System (C/C++)

A simple and efficient **console-based Stock Management System** developed using **C/C++ and File Handling**. This project helps manage inventory by allowing users to add, remove, search, and display stock items.

---

##  Features

-  Add new stock items
-  Remove existing items
-  Search items by item code
-  Display all stock records in tabular format
-  Persistent data storage using file handling (`inventory.txt`)

---

##  Technologies Used

- C++
- File Handling (fstream)
- Object-Oriented Programming (OOP)
- Standard Input/Output

##  How It Works

- All stock data is stored in a file named `inventory.txt`
- Each record is saved in the format:
- - The program reads and writes data directly from/to the file

---

## How to Run

1. Open the project in **Dev C++ / any C++ compiler**
2. Compile the `main.cpp` file
3. Run the program
4. Choose options from menu:
   - Add Item
   - Remove Item
   - Search Item
   - Display Items
